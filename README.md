## 🛠 Troubleshooting

- **Application won’t start:**  
  Make sure you run the `start-agent.bat` file from the USB drive. Running files directly from ZIP or another folder can cause errors.

- **AI services not responding:**  
  Check your internet connection. Claude Code and OpenAI Codex need online access to work.

- **Missing Node.js or errors about Node:**  
  This package includes Node.js bundled inside. Make sure you run the batch file from the USB folder, not from elsewhere.

- **Permission denied errors:**  
  Some public or locked computers limit USB execution. Try using a different PC or make sure your user account has basic run permissions.
